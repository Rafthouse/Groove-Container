# Audit: Harmony Organism Engine — Implementation Gap Analysis

## Date: 2026-06-17

### Verified by source code inspection, not reports.

---

## CRITICAL: Wheel C (Harmony) entirely absent from UI and audio

- [x] `src/engine/harmony.ts` exists — generates HarmonyEvent[]
- [x] `src/engine/genotype.ts` generates harmony events in `generateFromGenotypeImpl`
- [x] WheelC type in types.ts — optional on GrooveOrganism
- [  ] **`src/engine/audioEngine.ts`** — `tick()` only fires percussion + bass. **No harmony engine integration. Harmony is never played.**
- [  ] **`src/App.tsx`** — no Wheel C panel. No harmony grid, no harmony controls, no harmony edit, no harmony visualization.

## CRITICAL: Percussion tracks — ghost snare routing

- `persistence.test.ts` line 94-105 refers to `ghostPerc` track with voice `ghostPerc` that routes through its own mixer channel. The test expects a track named `ghost-snare` with voice `ghostPerc`.
- The current genotype generates only 6 tracks: kick(16), snare(16), closedHat(16), openHat(16), perc(16), ghostPerc(16) — all `cycleLength: 16`. No polymeter variety.
- UI renders all 6 tracks in Wheel A. The user reports "percussion track disappeared" — this may be a CSS/layout issue for 6 tracks.

## CRITICAL: No harmony audio rendering

- `audioEngine.ts` has `firePercussion()` and `fireBass()`. No `fireHarmony()` or equivalent.
- `HarmonyEvent` has `pitches[]` array with MIDI note numbers.
- Need a polyphonic harmony synth (multiple Tone.Synth or Tone.PolySynth).

## HIGH: Mutation Engine doesn't touch Wheel C

- `mutation.ts` mutates Wheel A (rhythm) events and Wheel B (bass) events. Wheel C events are untouched.
- After mutation, Wheel C may need regeneration via genotype + harmony engine.

## HIGH: Genotype UI incomplete

- GUI shows Genes panel via `showGenes` toggle.
- `GENOTYPE_OPTIONS` and `GENOTYPE_LABELS` in `genotype.ts` — verify they include all 21 genes (12 rhythm + 4 modal bass + 5 harmony).
- The select dropdowns render from `GENOTYPE_OPTIONS[key]`.

## HIGH: MIDI Export excludes Wheel C

- `src/engine/midi.ts` — exports percussion + bass. No harmony track.
- Need to add harmony channel output.

## MEDIUM: Event Detail Popup incomplete for harmony

- `EventDetailPopup` handles percussion and bass events. No harmony event editing UI.
- Harmony events need: root, inversion, velocity, pitches preview, duration.

## MEDIUM: Bass track volumes not editable in UI

- `WheelB` tracks show a volume slider but it's not connected to `audioEngine.setVoiceVolume()`.
- Pan slider not connected.

## MEDIUM: Presets

- `src/engine/presets.ts` — need to check if any presets have wheelC populated. If not, harmony is only available via genotype generation.
- Default presets should include harmony-enabled organisms.

## LOW: CSS for 6 tracks

- With 6 percussion tracks + 1 bass track + 1 harmony track = 8 track strips.
- Current CSS may not handle 8 tracks gracefully in the layout.
